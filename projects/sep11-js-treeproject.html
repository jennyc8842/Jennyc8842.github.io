Javascript Tree Project

Context

This project is primarily focused to practice loop(s), object(s), and other concepts I've learned so far in javascript from class.

This project involves working with an API dataset chosen from open data.

Working with my groupmates, Cami and Ivan, we used this dataset about trees.

What this project does is that it will ask you to put in a zipcode and it will output information about the trees that lives by your zipcode.

Challenges

One of the biggest challenges out there was the fact that there were A LOT of ideas on what we could do with the tree dataset we chose. Me and my groupmates kept on going back and forth on what we want to do, and it was certainly messy at times. We had 2 main ideas that we liked. One was for the user to put the zipcode of their choice and then the program would output what type of tree lives by that zipcode. The second one was for the user to input a certain type of tree and then the program will output where that tree will be. I didn't want to focus on too many ideas at once, so my group decided to stick with the first main idea and alter it as we go.

Another challenge is that sometimes me and my groupmates forget to add an extra = or } which generally are the main source of our errors. For example, there was at one point where our code had an error specifically on our if statement where if our API dataset has that zipcode, then it will put all the tree information into arrays. We didn't realize why that was happening. After some time analyzing it, we realized that our if statement had only one = sign which caused the error. All we needed to do was add another = to it (making it ==) and the error was gone.

Next Steps

Our next steps for this group project may be to add images of the trees whenever information about that specific tree pops up.

It would be also pretty neat to add accordions with titles of the tree name and then all the other information will pop out.

Also, adding colors and fancy fonts would be pretty cool as well.
